# 《wechaobishe》游戏介绍文档

## 🎮 游戏概述
《wechaobishe》是一款结合了**类幸存者（Survivor-like）**与**Roguelite 随机成长**玩法的网页端 2D 射击生存游戏。玩家将操控各具特色的 Emoji 角色，在潮水般的敌人围攻中生存，通过拾取经验宝石进行无上限的等级提升与能力抽卡，最终在“最终考验”中迎战拥有独立机制与动态数值的史诗级 Boss！

## 🎯 核心玩法特色

### ✨ 核心机制
- **割草与生存**：面对海量多体型的敌人潮，在无边际的空间内闪躲并依靠自动锁敌输出。
- **Roguelite 随机升级**：敌人死亡掉落不同品质的经验宝石（绿/蓝/黄/红）。升级后触发“三选一”随机强化（攻速、多重射击、伤害、生命、移速、磁铁），等级无上限！
- **特色专属能力**：不再只是简单的数值叠加，每个角色拥有独特的攻击弹幕特效，且大招充能完毕（默认30秒）会自动释放改变战局的终极技能。
- **史诗级最终考验**：游戏最后阶段将迎来强大的 Boss 战。Boss 具备专属华丽横幅预警、独立血条，且各项数值会根据玩家当前的 DPS 和血量**动态平衡**。Boss 战期间强制锁定 1 倍速，考验真正的走位与硬实力。
- **纯净的视觉与音效**：全新的左右对称护法面板（三列式全景 UI），沉浸式的大视野战斗，辅以打击粒子特效、屏幕飘字和丰富的游戏内音效。

### 🐉 敌人与 Boss 体系
**常规怪物（每 60 秒进化一个阶段，体型与血量波动生成）：**
1. **阶段0**：小绿怪 🦠 - 炮灰级，移速较慢。
2. **阶段1**：小恶魔 👿 - 移速提升，形成初步包围圈。
3. **阶段2**：丧尸 🧟 - 血量增厚，移动肉盾。
4. **阶段3**：吸血鬼 🧛 - 高智商寻路与高伤害机制。
5. **阶段4**：恶龙 🐉 - 体型巨大，压迫感极强。

**最终考验 Boss 库（动态数值、独立技能）：**
- **泰坦巨兽 🦍（近战）**：高血量猛兽。专属技能【泰山压顶】，会瞬间跳跃瞬移至玩家附近造成大范围巨额伤害。
- **虚空领主 👁️（远程）**：会主动风筝玩家。专属技能【虚空凝视】，释放会追踪玩家拐弯的锁定魔法弹幕。
- **混沌魔王 😈（混合）**：双修魔王。专属技能【混沌爆发】，释放 360 度全屏弹幕，并伴随一次致命冲锋。

## 🎮 操作与控制说明

游戏支持全键盘、全鼠标或键鼠混合的双模操控：

### 移动与战斗
- **键盘移动**：`W` / `A` / `S` / `D` 或 方向键 `↑` `↓` `←` `→`。
- **鼠标移动**：在游戏画面内**按住鼠标左键**不放，角色会自动朝着光标位置移动。
- **自动战斗**：角色自动锁定范围内的最近敌人进行射击，技能自动充能释放。

### 界面与快捷键
- **升级选择**：出现三选一卡片时，可点击鼠标，或用 `A` / `D` 切换高亮，按 `空格` (Space) 确认选择。
- **跳过升级**：按 `ESC` 或点击跳过按钮，可直接放弃本次升级。
- **调整倍速**：按 `Q` 减速，按 `E` 加速（支持 1x, 2x, 3x, 5x）。*注意：Boss战强制锁定 1x。*
- **游戏暂停**：按 `ESC` 唤出暂停菜单。

## 📊 升级能力池详解
每次角色升级，将从以下能力中随机出现 3 项供玩家选择：
1. 🏹 **攻速提升**：攻击间隔减少 20%。
2. ✨ **多重射击**：发射弹道数量 +1（呈扇形发射）。
3. 🗡️ **力量增强**：所有伤害提升 30%。
4. ❤️ **强健体魄**：最大生命值 +30，并立即恢复等量生命。
5. 🏃 **轻盈步伐**：移动速度数值 +20。
6. 🧲 **超级磁铁**：宝石拾取范围扩大 50%。

## 🎯 胜利与失败条件
- **胜利（标准模式）**：存活至指定时间（如 5 分钟），触发并击败“最终考验”的所有 Boss。
- **失败**：玩家角色生命值归零（被敌人触碰或被敌方弹幕击中）。
