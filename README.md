# 《wechaobishe》游戏介绍文档

## 🎮 游戏概述
《wechaobishe》是一款基于**类幸存者（Survivor-like）**玩法的网页端弹幕生存游戏。游戏名称源自网络谐音梗，结合“微嘲讽、避射击”的核心玩法理念，玩家将操控个性角色在潮水般的敌人围攻中生存，通过策略走位和攻击挺过不同难度阶段。

## 🎯 核心玩法特色

### ✨ 核心机制
- **弹幕生存**：面对逐渐增强的敌人潮，在有限空间内闪躲、攻击
- **角色成长**：实时属性提升，能力值蓄满可触发专属终极技能
- **阶段推进**：每60秒进入新阶段，解锁更强敌人类型
- **动态难度**：敌人生成频率和属性随阶段动态调整

### 🎪 角色系统
游戏提供3种独特角色，各有专长定位：

| 角色 | 表情 | 定位 | 属性侧重 | 专属技能 |
|------|------|------|----------|----------|
| 均衡小子 | 😀 | 全能型 | 生命100/伤害40/射程40 | 大治愈+增伤 |
| 疾风行者 | 😎 | 速射型 | 生命80/伤害20/射程80 | 射程突破+增伤 |
| 重装猛男 | 🤡 | 爆发型 | 生命100/伤害80/射程30 | 火力覆盖+增血 |

### 🎲 难度模式
4种难度选择，满足不同玩家需求：

| 难度 | 胜利时间 | 刷怪频率 | 怪物属性 | 适合人群 |
|------|----------|----------|----------|----------|
| 简单 | 5分钟 | 1.2倍 | 0.8倍 | 新手入门 |
| 中等 | 5分钟 | 1.0倍 | 1.0倍 | 标准体验 |
| 困难 | 10分钟 | 0.7倍 | 1.3倍 | 挑战极限 |
| 无尽 | 无限 | 0.6倍 | 1.0倍 | 持久战 |

### 🐉 敌人体系
敌人分为5个进化阶段，越后期越具威胁：

1. **阶段0**：小绿怪 🦠 - 基础敌人
2. **阶段1**：小恶魔 👿 - 移速提升
3. **阶段2**：丧尸 🧟 - 血量增厚
4. **阶段3**：吸血鬼 🧛 - 高智商追踪
5. **阶段4**：恶龙 🐉 - 终极Boss属性

## 🎮 操作说明

### 基础控制
- **移动**：WASD 或 方向键 (↑↓←→)
- **攻击**：自动锁定最近敌人
- **技能**：能力条蓄满自动释放

### 游戏控制
- **倍速调节**：1x/2x/3x/5x 四档速度
- **暂停/继续**：菜单按钮或游戏内暂停
- **重新开始**：暂停界面选择

## 📊 属性系统详解

### 角色属性公式
- **移动速度** = 100 + (移动值 ÷ 5) 像素/秒
- **攻击范围** = (范围值 + 20) × 5 像素
- **能力增长** = 2点/秒，100点触发终极技
- **攻击频率** = 角色专属攻击间隔

### 敌人AI系统
- **寻路逻辑**：基于智商值的周期性路径规划
- **攻击判定**：20 + (范围值 ÷ 10) 像素
- **移动速度**：20 + (移动值 ÷ 2) 像素/秒

## 📈 游戏平衡设计

### 前期体验（0-2分钟）
- 低血量敌人，快速清怪的割草感
- 角色基础属性足以应对初期压力
- 技能蓄能节奏平缓引入

### 中期压力（2-5分钟）
- 敌人类型混合，需要策略应对
- 刷怪频率提升，走位要求增加
- 专属技能成为生存关键

### 后期挑战（5+分钟）
- 精英敌人比例大幅增加
- 同屏敌人数量形成"尸潮"
- 资源管理和极限操作考验

## 🎯 胜利条件

### 标准模式
- 在限定时间内生存（简单/中等：5分钟，困难：10分钟）
- 击败不断进化的敌人波次
- 保持生命值高于零

### 无尽模式
- 挑战个人极限生存时间
- 追求最高阶段达成
- 竞争排行榜位置
